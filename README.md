## 概要
このブログは以下の技術を使用しています：
- **Next.js**: フロントエンドフレームワーク
- **Notion**: コンテンツ管理（ヘッドレスCMS）
- **Tailwind CSS**: スタイリング
- **TypeScript**: 型安全なコード
- **Netlify**: デプロイとホスティング
